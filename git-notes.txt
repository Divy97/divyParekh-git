1) ls :- list all the files and directories
2) mkdir :- make a directory 
3) cd :- change directory 
4) git init :- initialize empty repository 
5) ls -a :- show hidden files 
6) touch fileName :- to create a file 
7) git status :- track the changes in the repository
8) git add . :- add everything(every files and folders) into staging area
PS :- you also can write individual file or folder name instead of "."
9) git commit -m "MESSAGE" :- add a message to your commit
10) git restore --staged names.txt :- to restore the file from stage 
11) git log :- to see the history 
12) rm -rf names.txt :- to delete names.txt file
13) git reset hashID :- to reset commits
14) git stash :- move unwanted files to backStage 
15) git stash pop :- move files back to staging area
16) git stash clear :- to clear backstage 
17) git remote add origin https://github.com/Divy97/divyParekh-git.git :- adding github repository link
18) git remote -v :- show attached folders
19) git push origin master :- to push the changes into github repo
